## Расчёты



### Полный размер

$SIZE^A = \sum_{k=1}^{N} SIZE_k$

### PM для SCED

$PM^B = A*(SIZE^A)^E*SCED$

### Доля PM SCED для каждого компонента

$PM_k^B = \frac{PM^B * SIZE_k}{SIZE^A}$

### PM полная для каждого компонента

$PM'_k = PM_k^B * \prod_{i=1}^{16} EM_i$


### PM результат

$PM = \sum_{k=1}^N PM'_k$
